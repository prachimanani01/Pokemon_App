## Installation
To run this project locally, follow these steps:


1. **Clone the repository:**
    ```sh
    git clone https://github.com/prachimanani/pokemon_app.git
    ```


2. **Navigate to the project directory:**
    ```sh
    cd pokemon-app
    ```



    3. **Install dependencies:**
    ```sh
    flutter pub get
    ```


 4. **Run the app:**
    ```sh
    flutter run
    ```
